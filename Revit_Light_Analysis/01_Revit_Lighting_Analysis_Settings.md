# 01. Revit Lighting Analysis 설정 가이드

Revit의 **Lighting Analysis in the Cloud** 기능은 Autodesk 클라우드 서버를 활용하여 공간의 조도(Lux)를 시뮬레이션하는 도구입니다.

본 문서는 해석 설정 창의 각 항목별 의미, 실무 최적 설정값, 그리고 해석 데이터의 활용 방안을 정리합니다.

## 1. 설정 항목 상세 분석 (Settings Breakdown)

| 설정 항목 (Item) | 현재 설정값 (Default) | 의미 및 설명 | 실무 활용 팁 |
| :--- | :--- | :--- | :--- |
| **Analysis Type**<br>(해석 유형) | `Illuminance Analysis` | **공간의 밝기(Lux) 측정**<br>바닥이나 작업면에 빛이 얼마나 도달하는지 계산합니다. | 가장 기본적인 빛 환경 분석 도구입니다. |
| **Levels**<br>(분석 레벨) | `All` / `L1`... | **분석할 층 선택**<br>건물 전체(`All`) 또는 특정 층만 선택 가능합니다. | 시간이 부족할 땐 `L1`처럼 특정 층만 선택해 테스트하는 것이 효율적입니다. |
| **Date/Time**<br>(날짜/시간) | `9/21 9am - 3pm` | **태양 위치 기준일**<br>일반적으로 **춘분/추분(9월 21일)**을 기준으로 합니다. | **LEED/친환경 인증 표준**입니다.<br>1년 중 태양 고도가 가장 평균적인 날을 기준으로 평가합니다. |
| **Threshold**<br>(임계값) | Lower: `300 Lux`<br>Upper: `3000 Lux` | **합격/불합격 기준 범위**<br>• **300:** 사무 업무를 위한 최소 밝기<br>• **3000:** 눈부심(Glare)이 발생하는 한계선 | 보고서에서 이 범위 내에 들어오는 면적(%)을 자동으로 계산해 줍니다. |
| **Plane Height**<br>(작업면 높이) | `32 inches`<br>(약 81cm) | **빛을 측정하는 높이**<br>바닥(0)이 아닌 **책상 높이**에서 측정합니다. | 사람은 책상 위에서 일하기 때문에 바닥에서 띄워서 측정하는 것이 정확합니다. |
| **Resolution**<br>(해상도) | `12 inch grid`<br>(약 30cm) | **측정 센서 간격**<br>값이 작을수록 촘촘하고 정밀합니다. | 12인치는 매우 정밀한 수준이며, 해상도가 높을수록 해석 시간이 길어집니다. |

## 2. 변수 조절에 따른 변화 (What If?)

설정값 변경에 따른 시뮬레이션 결과의 변화를 이해하면 프로젝트 조건에 맞는 최적 설정이 가능합니다.

* **날짜를 12월(동지)로 변경 시:**
    * 태양 고도가 낮아져 실내 깊숙이 일사가 도달하나, 일조 시간 단축으로 전체 평균 조도는 하락할 수 있음
* **작업면 높이를 0 inch로 설정 시:**
    * 복도, 로비, 강당 등 작업 가구가 없는 공간의 바닥면 조도 분석에 적용
* **임계값(Lower)을 500 Lux로 상향 시:**
    * 정밀 작업(제도, 수술 등)이 요구되는 공간의 기준 적용. 합격 기준이 엄격해짐

<br>

## 3. 데이터 활용 및 기대 효과 (Utilization)

이 해석 데이터를 통해 생성된 **빛의 지도(Heatmap)**와 수치 데이터는 다음과 같이 활용됩니다.

### 친환경 인증 (LEED / G-SEED)
* **자연채광 확보** 항목의 증빙 자료로 활용하여 인증 점수(Credit) 획득
* 업무 시간 중 일정 비율 이상의 공간이 인공 조명 없이 300 Lux 이상을 유지함을 정량적으로 입증
* 전기 에너지 절감 효과 입증 근거로 활용

### 공간 배치 계획 (Space Planning)
* **눈부심 방지:** 조도 3,000 Lux 초과 구역에 모니터 배치를 지양하고 휴게 또는 비업무 공간으로 계획
* **조명 보완:** 자연광이 300 Lux 미달인 구역을 파악하여 인공 조명 배치 계획 수립

### 디지털 트윈 연동
* **쾌적도 시각화:** 각 실의 평균 조도값을 디지털 트윈 플랫폼에 입력하여 실내 환경 모니터링
* **시설 관리:** 조도 데이터 기반의 조명 교체 및 차양 제어 의사결정 근거 제공

---

> **참고:** 현재 설정된 값(`9/21`, `300~3000 Lux`, `32 inches`)은 오피스 빌딩 설계의 표준(Standard) 설정입니다.
