# Revit-Study-Personal

개인 학습용 Revit 정리 저장소입니다.

## 프로젝트 구조

```
├── docs/                          ← 웹 대시보드 (GitHub Pages 배포)
│   ├── index.html                    포털 페이지
│   ├── annual-building-systems/      연간 건물 에너지 분석 대시보드
│   └── hvac-dashboard/               HVAC 기계설비 분석 대시보드
│
├── Revit_Energy_Analysis/         ← 에너지 분석 보고서 & 리서치
│   ├── Report Data Files/            원본 데이터 및 분석 보고서
│   └── Report Settings Analysis Files/  시뮬레이션 세팅 분석 문서
│
├── Revit_Light_Analysis/          ← 조명 분석 리서치
│
├── images/                        ← 이미지 리소스
└── Zone-Equipment-Master-Guide.md ← 구역 장비 마스터 가이드
```

- **[HVAC Mechanical Analysis Dashboard](https://catsony5-web.github.io/Revit-Study-Personal/hvac-dashboard/)** — Revit 에너지 분석 대시보드
