# Revit 구역 장비(Zone Equipment) 마스터 가이드

이 문서는 Revit에서 구역(Zone)과 장비(Equipment)를 효율적으로 정의하고 관리하기 위한 기본 흐름과 체크리스트를 정리한 개인 학습용 가이드입니다.

## 1) 사전 준비

- 프로젝트 템플릿: MEP 템플릿 기준으로 시작
- 단위(Unit) 확인: 프로젝트 단위가 설계 기준과 일치하는지 점검
- 레벨(Level)과 뷰 템플릿: 구역/장비 작업용 전용 뷰를 미리 구성

## 2) 구역(Zone) 구성 흐름

1. 공간(Space) 배치
2. 공간 속성(이름, 번호, 면적, 부하 관련 파라미터) 정리
3. 구역 생성 및 공간 연결
4. 구역 요약표로 검토

## 3) 장비(Equipment) 배치 기준

- 장비 패밀리 타입과 용량 확인
- 기계실/층별 장비 그룹핑 기준 정의
- 장비가 연결되는 시스템(덕트/배관/전기) 사전 점검

## 4) 추천 파라미터(예시)

- Zone_Name: 구역 명칭
- Zone_Code: 구역 코드
- Space_Load: 공간 부하 값
- Equipment_Tag: 장비 태그
- Equipment_Capacity: 장비 용량

## 5) 품질 체크리스트

- 구역에 공간이 누락되지 않았는가
- 중복 장비가 배치되지 않았는가
- 장비 태그와 스케줄이 일치하는가
- 단위 변환 오류가 없는가

## 6) 출력 및 검토

- 구역/장비 스케줄을 내보내어 값 검증
- 검토용 도면 뷰를 PDF로 출력
- 변경 이력은 간단히 기록

## 7) 자주 발생하는 문제

- 공간 연결 누락 → 구역이 비어 보임
- 장비 타입 혼용 → 스케줄 값 불일치
- 단위 혼동 → 부하 값 과대/과소 산정

## 8) 개인 메모

- 템플릿에 구역 스케줄을 기본 포함
- 표준 장비 태그 규칙을 먼저 정의
- 필요 시 공유 파라미터로 확장
